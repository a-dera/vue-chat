# vuechat

## Instalation
```
git clone https://github.com/faso-dev/vue-chat.git
```
## Project setup
```
cd vue-chat && npm install
```

### Compiles and hot-reloads for development
```
npm serve
```

### Compiles and minifies for production
```
npm build
```

### Lints and fixes files
```
npm lint
```
